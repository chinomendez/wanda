## Grouping buttons

Sometimes you need to group buttons together to create a single visual context or to create a set of related actions. To group buttons, you can use the `ButtonsGroup` component.

export const exampleWithGroup = `<ButtonsGroup dimension="big" kind="secondary">
  <Button>Action 1</Button>
  <Button>Action 2</Button>
  <Button>Action 3</Button>
</ButtonsGroup>`;

<Components.Stack rowGap={24}>
  <Playground scope={{ ...Components }} code={exampleWithGroup} />
  <Components.Snackbar kind="info">
    <Markdown>
      The `ButtonsGroup` component overrides the buttons style and dimension.
      The style is always set to `secondary` while `dimension` is defined by the
      property applied on the group itself
    </Markdown>
  </Components.Snackbar>
</Components.Stack>
